Projeto: People Analytics - Previsão de Turnover
Este projeto visa a criação de um modelo preditivo para identificar colaboradores em risco de desligamento voluntário, utilizando técnicas de aprendizado de máquina.

Estrutura dos Notebooks
00_Extracao_Limpeza.ipynb: Pré-processamento e limpeza de dados.

Grupo 01 (Experimentos): Testes com diferentes modelos e técnicas:

01_2_LR_Standard_Under.ipynb

01_3_LR_Standard_Over.ipynb

01_4_LR_MinMax_SemBal.ipynb

01_5_LR_MinMax_Under.ipynb

01_6_LR_MinMax_Over.ipynb

01_7_SVM_Standard_SemBal.ipynb

01_8_SVM_Standard_Under.ipynb

01_9_SVM_Standard_Over.ipynb

01_10_SVM_MinMax_SemBal.ipynb

01_11_SVM_MinMax_Under.ipynb

01_12_SVM_MinMax_Over.ipynb

03_Feature_Importance.ipynb: Análise de importância das variáveis no modelo campeão.

04_Relatorio_Analise_Resultados.ipynb: Relatório final com diagnóstico e recomendações estratégicas.
